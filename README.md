### xdocs
xdocs can help generate blog posts

### setup
npm install xdocs -g

### usage
Usage: xdocs [Options]

选项：
  --server, -s   xdocs server                                             [布尔]
  --build, -b    xdocs build                                              [布尔]
  --clean, -c    xdocs clean                                              [布尔]
  --deploy, -d   xdocs deploy                                             [布尔]
  --version, -v  xdocs version                                            [布尔]
  --help, -h     显示帮助信息                                             [布尔]

示例：
  xdocs
  xdocs -s
  xdocs -v
